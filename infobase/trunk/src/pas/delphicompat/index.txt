title: DelphiCompat.pas
author: DanielPharos

This unit adds stuff that's missing in certain versions of Delphi. For instance, the function <tt>IsPathDelimiter</tt> was introduced in Delphi 6, but that would mean QuArK wouldn't be able to compile on Delphi 5 and lower, even though it could if only this function existed. To solve this problem, an implementation of this function has been added to this unit for these older versions of Delphi. (On version of Delphi that have this version, the build-in version is used.)

The version-definitions of Delphi all come from a single file: <tt>DelphiVer.inc</tt>, which is included through <tt>DelphiCompat.inc</tt>.

Then there are the version if-checks. Inside them the missing stuff is declared and defined. Some of it has been based on Delphi's own runtime source code. In a few rare cases, functionality that should have been in Delphi, but actually doesn't exist in any version, has also been added. And several Windows related functions and constants that were never added to Delphi are also defined here.

Note: All QuArK units that use any of the functions or constants that are declared here or use any of the constants in DelphiCompat, must have a DelphiCompat in their uses. Because this can easily be forgotten, breaking compilation on older versions of Delphi, it's added to every unit.
