title: OpenLogFile
author: DanielPharos

<code>
procedure OpenLogFile;
</code>

This procedure opens and clears/truncates the logfile. It creates a new file, or opens an existing file and blanks it. The filename is <tt>GetQPath(pQuArK)+LogFilename</tt>. It then writes the current date and time to the logfile.

It is safe to call this procedure if the logfile is already open; it will then do nothing.
