<b>Installation:</b>
<ol>
<li>Make sure, that you have at least the mini python package
   installed. Get it at the <a href="download.php">download page</a>.
   The full python install does it also (<a href="http://www.python.org">www.python.org</a>).
<li>Unpack the <a href="http://quark.sourceforge.net/infobase/glossary/index.html#.ACE">.ACE</a> archive with e.g. xaceplus from <a href="http://www.winace.com">www.winace.com</a>.
   A directory will be created.
<li>Start QuArK by clicking quark.exe.
<li>Don't forget to <a href="http://quark.sourceforge.net/infobase/configuration/index.html">configure</a> the Game directories and tools.
   If you don't have the map compiler tools for the game you are
   making maps for, please refer to the <a href="download.php">download page</a>.
<li>When making maps for Q3, check if you have the mapmedia.pk3
   file (from QERadiant) in your baseq3 directory. Otherwise
   some shader textures and models are missing.
<li>Please, when <a href="forums.php">reporting bugs</a>, state for which game the report
   is. Often it is game specific.
</ol>
