<headline>Zoner's Half-Life Tools v2.0.6</headline>
From the changelog:
<ul>
<li>Fixed an extremely subtle bug which would occasionaly crash vis when it was trying to compress the vis data before exiting.
<li>Added CLIP brushes to be allowable in entities. As long as there is at least one normal brush included in the entity, CLIP's may be included too.
<li>Fixed a crash which would occur occasionaly when an ORIGIN brush is placed in the world (ironically enough, the function printing the error message was causing it).
<li>Enabled some long-disabled code, the -incremental switch for hlrad, so that turnaround time for light tweaking can be reduced.
</ul>
Get the <a target="_blank" href="http://www.winace.com">WinAce</a> file of Zoner's Half-Life Tools: <a href="http://dl.fileplanet.com/dl/dl.asp?quark/zhlt206.ace">zhlt206.ace</a> (316KB), or
the <a target="_blank" href="http://www.winzip.com">WinZip</a> one (371KB) which can be found at the <a target="_blank" href="http://halflife.gamedesign.net">Half-Life ERC</a> site.<br>
