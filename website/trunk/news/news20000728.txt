This snapshot contains fixes and extensions to Q3/Bezier functions,
and fixes about HL support. It contains the same changes as snapshot
0725, plus one fix for the bezier bevel/arch macro.<br>
<br>
INSTALLATION:<br>
<ol>
<li>Make sure, that you have at least the mini python package
   installed. Get it at <a href="download.php">download page</a>.<br>
   The full python install does it also (<a href="http://www.python.org">www.python.org</a>)
<li>Unpack the <a href="help/glossary/index.html#.ACE">.ACE</a> archive with e.g. xaceplus from <a href="http://www.winace.com">www.winace.com</a>
   A directory will be created.
<li>Start QuArK by clicking quark.exe
<li>Don't forget to <a href="help/configuration/index.html">configure</a> the Game directories and tools. <br>
   If you don't have the map compiler tools for the game you are
   making maps for, please refer to <a href="download.php">download page</a>.
<li>When making maps for Q3, check if you have the mapmedia.pk3
   file (from QERadiant) in your baseq3 directory. Otherwise
   some shader textures and models are missing.
<li>Please, when reporting bugs, state for which game the report
   is. Often it is game <a href="forums.php">game specific</a>.
</ol>
Unpacking the snapshot over an older release may work at the first
glance, but may lead to subtile errors that are hard to find.<br>
I do not recommend it.<br>
<br>
