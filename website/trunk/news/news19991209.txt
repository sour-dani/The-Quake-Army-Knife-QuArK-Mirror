<headline>Zoner's Half-Life Tools v1.5</headline>
A new version of the tools for Half-Life map-building made by Zoner. Only the HLRAD program seems to have been modified:
<ul>
<li>Fixed BSP warning accidentally removed and replaced with a vague generic message. 'NULL Pointer' caused by a CLIP textured brush based entity now correctly logs an error.
<li>More documentation fixes.
<li>Cleaned up source code directory layout.
<li>Fixed Unix source package not having README and INSTALL text files.
</ul>
