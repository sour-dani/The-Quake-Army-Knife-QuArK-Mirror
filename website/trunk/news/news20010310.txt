<headline>QuArK v6.2 Release Candidate</headline>
<p>Here is the <a href="http://dl.fileplanet.com/dl/dl.asp?quark/QuArK_v6.2.zip">QuArK v6.2 release candidate</a>. The install instructions are as usual:</p>

<ul>
<li>If you haven't installed Python yet, do that using either the small <a href="http://dl.fileplanet.com/dl/dl.asp?quark/minipy15b.exe">minipy15b.exe</a> or get the larger Python v1.5 from <a href="http://www.python.org">www.python.org</a>.
<li>Unzip QuArK_v6.2.zip into a <b>new</b> and <b>empty</b> directory. <b>Do not</b> use your existing QuArK directory, since this may conflict with this release candidate.
<li>Start QuArK 6.2 and configure the build-tools etc.
</ul>
