<headline>Full release of QuArK 6.4.0 alpha 3</headline>
<p>Peter has made a full release through the SourceForge Release System of <a href="http://sourceforge.net/project/showfiles.php?group_id=1181&release_id=188980">QuArK 6.4.0 alpha 3</a> <span class=sml>(dated 2004-01-16)</span>.</p>

<p>Some of the items and features covered by this release are:</p>
<UL>
<LI>Three new shapes added to Basic Polyhedrons and standardization of all the Basic shape sizes and labeling for easy use with one another.
<LI>Instructions for installing QuArK on GNU/Linux.
<LI>New HotKey (Alt-D) and Selection menu function, Make Detail for setting multiple poly and face texture flag to detail and related Infobase update.
<LI>Numerous improvements and corrections by SilverPaladin (see NEWS.txt file included).
<LI>Removed reliance on external string library (QuArK now uses Python's built-in string library directly).
<LI>Python 1.5 no longer supported: QuArK now only works with Python 2.0 or higher.
<LI>Added 'Grid scale in 2D views' feature to Options menu. By activating the sub-menu items they will place the current grid setting numbers on that particular view and will increase or decrease as you zoom in and out.
</UL>

<p>The latest up dated Infobase (also included in release) and the source code can be downloaded seperately from the same link above.</p>

<p>For more information and support, go to the <a href="http://quark.sourceforge.net/forums.php">forums</a>.</p>
